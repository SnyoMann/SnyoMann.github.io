Selamat datang di SnyoMann.github.io! Ini adalah situs pribadi yang berfungsi sebagai portofolio dan blog untuk berbagi pengetahuan, proyek, dan pengalaman.

Tentang Saya
Halo! Saya SnyoMann, seorang pengembang perangkat lunak yang bersemangat tentang teknologi dan pemrograman. Di situs ini, saya berbagi proyek-proyek terbaru, tulisan-tulisan teknis, dan berbagai tutorial.

Konten Situs
Portofolio: Di bagian ini, Anda dapat menemukan berbagai proyek yang telah saya kerjakan, termasuk deskripsi, teknologi yang digunakan, dan tautan ke repositori GitHub terkait.
Blog: Bagian ini berisi artikel-artikel yang saya tulis tentang berbagai topik, mulai dari pengembangan perangkat lunak, tips dan trik pemrograman, hingga pembelajaran mesin dan AI.

Tentang Saya: Halaman ini berisi informasi lebih lanjut tentang latar belakang saya, keterampilan, dan pengalaman profesional.

Kontak: Jika Anda ingin menghubungi saya, Anda dapat menemukan informasi kontak saya di halaman ini, termasuk tautan ke profil media sosial saya.
Teknologi yang Digunakan

HTML/CSS: Untuk struktur dan tampilan dasar situs.

JavaScript: Untuk interaktivitas dan fungsionalitas dinamis.

Jekyll: Situs ini dibangun menggunakan Jekyll, sebuah generator situs statis yang memungkinkan saya untuk mengelola konten dengan mudah.
Cara Menggunakan

Navigasi: Gunakan menu navigasi di bagian atas halaman untuk berpindah antara halaman portofolio, blog, tentang saya, dan kontak.

Proyek: Di halaman portofolio, klik pada proyek untuk melihat detail lebih lanjut.

Artikel: Di halaman blog, klik pada judul artikel untuk membaca tulisan lengkap.
